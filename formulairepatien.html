Bienvenue sur FertiConnect votre plateforme de santé reproductive

Nous vous remercions de bien vouloir prendre quelques minutes pour remplir ce formulaire. Vos réponses nous aideront à mieux comprendre votre situation personnelle et médicale afin de vous offrir un soutien personnalisé. Toutes les informations que vous fournissez resteront strictement confidentielles.


Informations Personnelles

Indiquez votre date de naissance.

Sélectionnez votre sexe.

Entrez votre adresse complète.

Indiquez votre numéro de téléphone.

Informations Médicales

Indiquez votre âge actuel.

Sélectionnez votre état civil actuel.

Indiquez le nombre d'enfants que vous avez.

Indiquez si vous souhaitez avoir plus d'enfants.

Veuillez insérer l'ID de votre partenaire. Vous pouvez trouver cet ID dans le profil de votre partenaire. L'ID comporte 28 caractères et est au format suivant : "xxxxxxxxxxxxxxxxxxxxxxxxxxxx".

Décrivez brièvement vos antécédents médicaux importants.

Indiquez si vous avez des allergies connues et lesquelles.

Listez les médicaments ou compléments alimentaires que vous prenez actuellement.

Indiquez si vous avez déjà rencontré des problèmes de fertilité.

Indiquez si vous avez été diagnostiqué avec des troubles hormonaux.

Informations Complémentaires

Indiquez si vous avez des antécédents de maladies sexuellement transmissibles.

Indiquez si votre cycle menstruel est régulier.

Indiquez si vous avez déjà eu des complications lors de grossesses précédentes.

Précisez le trimestre de l'avortement le cas échéant.

Indiquez votre profession actuelle.

Indiquez si vous êtes exposés à des pesticides ou des températures importantes dans votre travail.

Indiquez si vous travaillez dans des conditions stressantes.

Indiquez si vous êtes fumeur.

Indiquez le nombre de cigarettes que vous fumez par jour, le cas échéant.

Indiquez si vous consommez de l'alcool.

Indiquez votre consommation d'alcool hebdomadaire, le cas échéant.

Indiquez si vous consommez du cannabis.

Indiquez votre consommation hebdomadaire de cannabis, le cas échéant.

Indiquez si vous pratiquez une activité physique régulière.

Précisez quelle activité physique vous pratiquez et à quelle fréquence, le cas échéant.

Décrivez vos objectifs en matière de santé reproductive.

Décrivez ce que vous attendez de cette plateforme pour vous aider dans votre parcours de santé reproductive.
